# Data Engineering Assignment

## Objective
Perform basic data exploration and cleaning using Pandas.

## Files Included

- Data_Engineering_Assignment_Final_Submission.ipynb
- Combined_dataset.csv
- cleaned_combined_dataset.csv
- Assignment_Summary_Updated.docx

## Operations Performed

- Loaded dataset using Pandas
- Explored data using head(), tail(), shape, columns, and dtypes
- Checked missing values
- Handled missing values using fillna()
- Selected important columns
- Filtered products with rating greater than 4
- Removed duplicate records
- Created a new column: rating_score
- Saved cleaned dataset

## Tools Used

- Python
- Pandas
- Jupyter Notebook
